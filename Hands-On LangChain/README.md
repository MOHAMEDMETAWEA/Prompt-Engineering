# Hands-On LangChain

## Folder Purpose
Contains practical LangChain notebooks for document loading, retrieval, and evaluation.

## File Structure
- `requirements.txt` - Python dependencies used by this tutorial path.
- Multiple `.ipynb` notebooks: core techniques such as document loading, chains, vector stores, question answering.

## Data Flow
1. Data ingestion in `Documents_Loading_with_LangChain.ipynb`.
2. Preprocessing and splitting in `LangChain_Document_Splitting.ipynb`.
3. Vector store build and query in `LangChain_VectorStores_Embeddings.ipynb`.
4. RAG pipeline end-to-end in `RAG_Pipeline_with_LangChain.ipynb`.

## Key Components
- Loader chains
- Retriever and Vectorstore adapters
- Prompt templates
- Evaluation notebook

## Usage Instructions
1. Create virtual environment.
2. Install dependencies: `pip install -r requirements.txt`.
3. Open notebooks in Jupyter and run sequentially.
