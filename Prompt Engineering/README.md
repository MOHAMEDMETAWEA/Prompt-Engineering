# Prompt Engineering

## Folder Purpose
This folder contains foundational notebooks and resources for prompt engineering techniques including few-shot, templates, memory, output parsing and an introduction to LLMs.

## File Structure
- `01-LLM-models.ipynb`, `02_prompt_Templates.ipynb`, etc.: sequential lessons.
- `chapter3.ipynb`...`chapter9.ipynb`: course chapters.
- `youtube-sub.csv`: sample dataset.

## Data Flow
Notebooks are structured for learning; each one is mostly self-contained, with later chapters building on earlier concepts.

## Key Components
- Prompt template design
- Conversational memory strategies
- Output parser validation
- Retrieval-augmented generation (RAG)

## Usage Instructions
1. Open notebooks in an environment supporting Jupyter.
2. Run each notebook in order for progressive learning.
3. Add your own datasets and prompt experiments as you iterate.
